# Color-character-output
-----中文-----
本项目实现了便捷的在终端输出彩色字符。

"彩色输出.cpp"对项目进行了演示，你可以直接进行编译并运行。
"RGB.cpp","RGB.h","colorOut.cpp","colorOut.h"四个文件是本项目的核心。

类RGB作为参数保存使用。
类colorOut作为本项目的核心，内部实现了类的构造和输出，同时现在它还支持链式编程。

本项目为创建者练习使用
